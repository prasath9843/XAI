# Kidney Disease Prediction using SHAP and LIME

## Overview

This project predicts Chronic Kidney Disease (CKD) using a Random Forest Classifier and explains predictions using SHAP and LIME.

## Features

- Data preprocessing
- Missing value handling
- Label encoding
- Random Forest Classification
- SHAP Explainability
- LIME Explainability
- Feature Importance Visualization

## Dataset

Kidney Disease Dataset

## Libraries Used

- Pandas
- NumPy
- Scikit-Learn
- SHAP
- LIME
- Matplotlib

## Results

- Classification Accuracy
- SHAP Summary Plot
- SHAP Feature Importance Plot
- LIME Local Explanation
